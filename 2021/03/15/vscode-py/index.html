python 配置

    先下载

Python！点击图中的Downloads。

    选择最新版本3.9.2。

    以下就是py安装程序辣！安装的时候，注意要选择最后一个Add Python 3.9 to Path。然后单击Install Now。

    默默等待其安装完成。

    在其安装完成之后，打开bash或cmd，输入python -V来查看py版本，出现版本号，则说明安装成功！

接下来：

  1. 在cmd输入pip list检查是否已存在需要的包；
  2. 输入pip install flake8下载安装flake8；
  3. 输入pip install yapf下载安装yapf。
